# Azure Cognitive Services

**Azure Congnitive Services** is a suite of pre-built AI services that enable developers to add intelligent features to their applications without requiring any expertise in machine learning or data science. These services include: 
- Computer Vision
- Speech
- Language
- Decision

**Azure Congnitive Services** is a cloud-based service that enables organization to build, train and deploy machine learning models at scale. With Azure Machine Learning, data scientists and developers can: 
- Build machine learning models
- Train models
- Deploy models 

**Azure Databricks** is a collaborative, cloud-based platform for data engineering, machine learning, and analytics. It is based on Apache Spark and provides a unified analytics experience for data scientists, data engineers, and business analysts. With Azure Databricks, you can: 
- Build and train machine learning models
- Analyse large datasets
- Collaborate with team members

**Azure Cognitive Search** is a fully managed search service that enables organizations to add search capabilties to their applications with ease. With Azure Cognitive Search, you can: 
- Index and search structured and unstructured data from various sources. 
- Cusotmize search results
- Scale up or down based on demand and monitor search performance. 

**Azure AI Solutions** are a suite of AI services and tools offered by Microsoft Azure. 
These solutions include: 
- Azure Cognitive Services
- Azure Machine Learning 
- Azure Databricks
- Azure Cognitive Search
